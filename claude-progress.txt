## HOW TO RUN

```bash
# Setup (first time)
chmod +x init.sh
./init.sh

# Or manually:
cd backend && npm install && npm run seed
cd frontend && npm install

# Development
# Terminal 1:
npm run dev --prefix backend

# Terminal 2:
npm run dev --prefix frontend

# Access
# Frontend: http://localhost:5173
# Backend: http://localhost:3001
```
