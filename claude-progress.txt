## PROJECT OVERVIEW

### Application Features
The JIVS Migration Visual Companion is a fully-featured dashboard application that transforms complex migration data into intuitive visualizations:

1. **User Authentication** - Login, registration, password reset, session management
2. **Dashboard Views** - Reconciliation overview with charts and statistics
3. **Custom Dashboards** - Create, edit, delete, duplicate dashboards
4. **Visualizations** - Multiple chart types with template library
5. **Test Rules View** - Problem-first display with grouping and filtering
6. **Comparison View** - Side-by-side run comparison with diff highlighting
7. **Export Functionality** - PDF, CSV, JSON exports
8. **Notifications** - In-app notification system with alerts
9. **Theme Support** - Dark/Light mode with smooth transitions
10. **Beginner Support** - Tooltips, help panels, info icons
11. **Admin Features** - User management for admin role

### Technical Stack
- **Frontend**: React + Vite, Tailwind CSS, Framer Motion, Recharts
- **Backend**: Node.js + Express, SQLite database
- **Authentication**: JWT-based with session management
- **Data**: 3,976 reconciliation records, 4,399 test rule records

## SERVER CONFIGURATION

The current setup uses:
- **Backend**: Port 3001 (or 3002 if conflict)
- **Frontend**: Port 5173 or 5174
- **Frontend proxy**: Targets backend port

To start servers:
```bash
npm run dev --prefix backend   # Starts backend
npm run dev --prefix frontend  # Starts frontend
```

Access the app at: http://localhost:5173 or http://localhost:5174
