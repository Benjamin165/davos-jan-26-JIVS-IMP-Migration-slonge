## IMPORTANT: SERVER CONFIGURATION

Due to conflicts with old server processes, the current setup uses:
- **Backend**: Port 3002 (configured in backend/.env)
- **Frontend**: Port 5173 or 5174
- **Frontend proxy**: Targets localhost:3002

To start servers:
```bash
npm run dev --prefix backend   # Starts on port 3002
npm run dev --prefix frontend  # Will use 5173 or 5174
```

Access the app at: http://localhost:5173 or http://localhost:5174
