## HOW TO RUN
```bash
# Setup (first time)
chmod +x init.sh
./init.sh

# Or manually:
cd backend && npm install && npm run seed
cd frontend && npm install

# Development
# Terminal 1:
cd backend && npm run dev

# Terminal 2:
cd frontend && npm run dev

# Access
# Frontend: http://localhost:5173
# Backend: http://localhost:3001
```
